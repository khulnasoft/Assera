# Assera v1.2 Reference

## `assera.Schedule.get_indices()`
Gets the iteration space dimensions for a schedule.

## Returns
Tuple of `Index`

## Examples

Get the iteration space dimensions for a 3-dimensional nest:

```python
i, j, k = schedule.get_indices()
```
