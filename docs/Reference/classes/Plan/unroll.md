# Assera v1.2 Reference

## `assera.Plan.unroll(index)`
Marks a dimension of the iteration-space for unrolling.

## Arguments

argument | description | type/default
--- | --- | ---
`index` | The index to unroll. | `Index`

## Examples

Mark the `i` dimension for unrolling:

```python
plan.unroll(index=i)
```
